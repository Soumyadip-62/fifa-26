# DESIGN_SCOPE.md

## Current Design Approach

The frontend should now move from a basic layout to a more modern, attractive, sports-dashboard style UI.

Design should feel polished enough for a football stats/news app, but should still remain practical, maintainable, and not over-engineered.

Use Tailwind CSS and shadcn/ui as the default design system.

The app should look good in both light mode and dark mode.

## Design Direction

The UI should feel like a modern FIFA / football tournament dashboard:

- Clean
- Fast
- Sports-focused
- Visual-first
- Dark-mode friendly
- Card-based
- Responsive
- Easy to scan
- Slightly premium, but not flashy

The design should use football visuals where available:

```txt
team logos
country flags
player images
stadium images
match banners
news/blog cover images
